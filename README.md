# 🌍 KindBeings NGO Website

A responsive and user-friendly NGO website developed using HTML and CSS to spread awareness, encourage volunteering, and support social causes related to women empowerment and community welfare.

---

# 🚀 Project Overview

The KindBeings NGO Website is designed to provide information about the organization, its mission, projects, volunteering opportunities, and contact details.

The website aims to create awareness and help people connect with social initiatives through a simple and clean interface.

---

# ✨ Features

- Responsive website design
- Home page with NGO introduction
- About Us section
- Volunteer page
- Women empowerment section
- Projects page
- Contact page
- Clean and simple UI

---

# 🛠️ Tech Stack

- HTML5
- CSS3

# ▶️ How to Run

1. Download or clone the repository  
2. Open the project folder
